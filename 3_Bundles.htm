Symfony 5

3.- Bundles

IMPORTANTE!!!  Comprobar que se está trabajando en la carpeta del proyecto

Un bundle es un paquete de archivos que ofrecen una funcionalidad a nuestra app

Mostrar listado de bundles instalados

composer recipes

Instalación con Composer

composer require vendor/nombre-bundle --dev

--dev Instalar solo para desarrollo

maker-bundle

Generador de código

https://symfony.com/doc/current/bundles/SymfonyMakerBundle/index.html

composer require symfony/maker-bundle --dev

annotations

Añade anotaciones a las clases

https://www.doctrine-project.org/projects/doctrine-annotations/en/1.10/index.html

composer require doctrine/annotations

profiler

Añade un toolbar y el método dump() (equivalente a var_dump())

composer require profiler --dev
